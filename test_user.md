## Scenario pas a pas

1) Connexion admin
   - Creer un coordinateur d'apprentissage.
   - Creer un responsable de formation.
   - Creer une promotion (E5a - Quere).
   - Creer un apprenti.
   - Synchroniser la promotion.
   - Creer un maitre d'apprentissage.
   - Creer un tuteur pedagogique.
   - Creer un professeur.
   - Creer un intervenant.
   - Creer une entreprise.
   - Associer tuteur + maitre a l'apprenti.
   - Associer l'apprenti a la promotion.
   - Associer l'entreprise a l'apprenti.
   - Creer un jury de test pour l'apprenti.

2) Connexion coordinatrice
   - Voir le calendrier avec les echeances de tous les apprentis.
   - Ouvrir un journal d'apprenti.
   - Consulter les entretiens d'un apprenti.
   - Creer un jury (selectionner les membres).
   - Creer/modifier les promotions et semestres.

3) Connexion apprenti
   - Voir le calendrier avec ses echeances.
   - Deposer un document (notes mensuelles ou rapport).
   - Creer un entretien (choisir un semestre + date dans le semestre).
   - Consulter la page Jury.

4) Connexion tuteur pedagogique
   - Voir le calendrier des apprentis suivis.
   - Ouvrir le journal d'un apprenti.
   - Telecharger un document.
   - Ajouter un commentaire sur un document.
   - Consulter les entretiens.
   - Valider/refuser la date d'entretien.
   - Donner une note a un entretien.
   - Consulter les jurys de ses apprentis.

5) Connexion maitre d'apprentissage
   - Voir le calendrier des apprentis suivis.
   - Ouvrir le journal d'un apprenti.
   - Telecharger un document.
   - Ajouter un commentaire sur un document.
   - Consulter les entretiens.
   - Valider/refuser la date d'entretien.
   - Renseigner les competences du semestre en cours.

6) Connexion professeur / intervenant
   - Consulter les jurys associes.
   - Consulter les documents du jury.
   - Saisir la note du jury.
